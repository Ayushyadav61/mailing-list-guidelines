Mailing List Guidelines 1.0.0

Here are some pointers which you need to keep in mind:

  1. Always use a search engine before you ask a question.
  2. DO NOT use CAPITAL LETTERS as it is like shouting.
  3. Do NOT post questions with No Subject or very long Subject line.
  4. Do NOT send HTML mails to mailing list. Also don't send Dual-format mails containing both text and html.
  5. DO NOT use SMS Texts like 'u' for 'You' etc.
  6. DO NOT hijack discussion thread, always start a fresh thread for discussion something new.
  7. Trim properly while replying , follow interleaved or bottom style of posting to the list.
  8. Use [Commercial] tag in subject line for mails related to Jobs.
  9. Use [OT] tag in subject line for any info which is Off Topic.
  10. Avoid cross posting.
  11. Avoid sending attachments to the mailing list.
  12. Use http://gist.github.com/ for posting Output or Error log.
  13. Download Shakthi Kannan's Presentation here.
  14. How to Ask Questions http://www.catb.org/~esr/faqs/smart-questions.html
